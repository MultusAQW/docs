# Equipped Items

## Base URL
```python copy
https://multus-aqw-api.vercel.app/api/aqw/items/equipped?name=Alina
```

## Query Parameters
| Parameter |  Type  | Description                | Required? | Default |
| :-------: | :----: | -------------------------- | :-------: | :-----: |
|   name    | string | The name/ign of a Character|    Yes    |  `""`   |

## Request Samples

  <Tab>
    <>
        ```js
        import axios from "axios";

        // Using "Alina" as name query
        const name = "Alina"
        const url = `https://multus-aqw-api.vercel.app/api/aqw/items/equipped?name=${name}`;
        const data = async () => {
            try {
                const { data } = await axios.get(url);
                return data;
            } catch (err) {
                throw new Error(err.message);
            }
        };

        console.log(data);
        ```
