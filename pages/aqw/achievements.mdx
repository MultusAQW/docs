# Achievements / Badges

## Base URL
```python copy
https://multus-aqw-api.vercel.app/api/aqw/items?name=Alina
```

## Query Parameters
| Parameter |  Type  | Description                | Required? | Default |
| :-------: | :----: | -------------------------- | :-------: | :-----: |
|   name    | string | The name/ign of a Character|    Yes    |  `""`   |

## Request Samples

<Tabs items={["JavaScript", "Python"]}>
  <Tab>
    <>
        ```js
        import axios from "axios";

        // Using "Alina" as name query
        const name = "Alina"
        const url = `https://multus-aqw-api.vercel.app/api/aqw/achievements?name=${name}`;
        const data = async () => {
            try {
                const { data } = await axios.get(url);
                return data;
            } catch (err) {
                throw new Error(err.message);
            }
        };

        console.log(data);
        ```
    </>

  </Tab>

    <>

        name = Alina
        url = f"https://multus-aqw-api.vercel.app/api/aqw/achievements?name={name}"

        data = response.json()
